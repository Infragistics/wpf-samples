カスタマイズ可能なレイアウト

作成しているアプリケーションにぴったりな凡例のレイアウト、ナビゲーション ペイン、その他すべてのマップ側面、マップ要素をデザインします。

SQL Server の地理空間データ

直接データベースからシェープ データを読み込んでネイティブ SQL Server 2008 の GEOMETRY および GEOGRAPHY タイプを使用できます。データをバインディングして追加の SQL フィールドをマップ シェープにマージすることもできます。レイヤーを使って SQL Server の地理空間シェープを 1 つのマップ コントロールにマージすることも可能です。

カラーモデル

複雑な色の見本やコロプレス シェーディングによって、都道府県や国、雨量、その他の業務データで地域を色分けするなど、多くの一般的なマップ シナリオを実現できます。
