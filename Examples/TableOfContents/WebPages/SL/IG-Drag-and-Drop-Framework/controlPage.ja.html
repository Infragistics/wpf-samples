Silverlight RIA の UI 要素でドラッグ アンド ドロップ動作を可能にする Infragistics® Drag and Drop Framework  は、ドラッグ アンド ドロップ エクスペリエンスの完全な制御、詳細なスタイル設定が可能です。

ドラッグ アンド ドロップ

Silverlight アプリケーションの任意の UIElement™ にドラッグ アンド ドロップ動作に追加することができます。
サンプルの表示 

UI 要素のドラッグ

任意の項目をドラッグし、定義したターゲットにドロップできます。
サンプルの表示 

 
 

チャネル

ドラッグ ソースおよびドロップ ターゲットを「チャネル」と関連付けて制約を定義できます。項目は一致するチャネルがある位置のみにドロップできます。
サンプルの表示 

カーソルのカスタマイズ

ドラッグ操作で表示される移動、コピー、および「ドロップ禁止」などのカーソルの外観をカスタマイズできます。
サンプルの表示 

 
 

ドロップ ターゲット マーカー

このターゲットでドラッグされている項目が有効な場合、スタイル可能で半透明なマーカーを描画します。
サンプルの表示 

 

イベント

複数のイベントによって操作の詳細をコントロールすることができます。
