解析

構文解析エンジンはテキストを処理し、エンジンに提供した文法のルールに基づいてトークンを作成します。読み込み処理で生成されたトークンは、指定した言語に基づいてドキュメント テキストに構文解析を提供するために使用されます。パーサーは決定的および非決定的文法を処理する高いパフォーマンス エンジンです。

ドキュメントの検索

構文解析エンジンは、Find、FindReplace、FindAll、FindReplaceAll のメソッドを使用して、ドキュメントで条件と一致するテキストのインスタンスを検索する機能を提供します。方向 (前方または後方)、大文字と小文字の区別、単語の全体または一部、検索するテキスト、RegEx パターンを使用して検索を指定できます。

構文ツリー / エラーの警告

構文解析エンジンは、エンジンに提供した言語のテキスト ルールの構文構造を表すテキスト ドキュメントの構文ツリーを公開します。構文ツリーは、エラーとして無効なノードをマークします。API で公開されたメソッドによってアクセスできます。

ドキュメント ナビゲーション

シンプルな API を使用して行、トークン単語をスキャンします。ドキュメントの開始または終了に移動し、特定の行、トークン、単語に移動できます。ドキュメントの現在の位置を変更せずに以前 / 次のトークンまたは単語を読み込むことができます。

EBNF のサポート

構文ライブラリは言語の EBNF でフィードされ、解析されてキーワード、トークンおよび言語構成を認識できます。プレーン テキスト、C#、VB の言語がサポートされています。ISO EBNF 表記 (ISO-14977:1996(E)) を完全にサポートします。
