Syntax Editor は、Microsoft Visual Studio のコード エディターのようなコード編集エクスペリエンスを提供するテキスト編集コントロールです。新しい構文解析エンジンで作成されました。言語固有の文法ルールを設定すると、言語の部分を表示し、テキストが構文ルールに合わせない場合にエラー情報を表示します。

編集サポート

Syntax Editor はコード エディターの編集機能を含みます。クリップボード サポート、元に戻すとやり直し、ドラッグ アンド ドロップ、行番号、文字と行選択をサポートします。

構文のハイライト表示

Syntax Editor は、コメント、キーワード、言語固有のエンティティを識別するために、テキストの解析の色などのビジュアル キューを表示します。複数のフォント スタイル (太字、イタリックなど) を 1 つのドキュメントに表示できます。
サンプルの表示 

 

カスタム言語のサポート

C# および VB のコード言語がサポートされています。カスタム言語が EBNF 文法定義ファイルを作成して使用できます。
サンプルの表示 

 

分割ビュー

Syntax Editor は、エディターの分割をサポートします。ユーザー インターフェイスまたはコードで分割できます。水平および垂直両方の分割がサポートされます。
サンプルの表示 

 

カスタム要素スタイル設定

Syntax Editor のエディター要素のテンプレートを変更せずに、要素をカスタマイズできます。定義済みの行番号マージンを提供し、xamSyntaxEditor コントロールの 4 つのマージン領域に配置可能なカスタム マージンを作成できます。カレット、行番号マージン、選択、背景、背景、フォント サイズ、フォント ファミリなどのエディター要素の外観をカスタマイズできます。言語の構文と文法ルールに基づいて、テキストの色設定 (構文強調表示) を管理できます。
