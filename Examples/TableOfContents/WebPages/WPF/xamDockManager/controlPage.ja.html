Visual Studio ドッキング コントロール

Visual Studio のドッキングのようにレイアウト永続化、フォルーティング ウィンドウ、および HWind サポートの機能を提供します。
サンプルの表示 

カスタマイズ可能なドッキング インジケーター

ドッキング可能な項目が固定する位置を示すビジュアル インジケーターをサポートします。
サンプルの表示 

 

コンテンツ ペイン ナビゲーター

コンテンツ ペインの間にナビゲートするプレビュー付きのナビゲーション画面をサポートします。
サンプルの表示 

 
 

タブ付きドキュメント

タブ付きの MDI のようなタブ領域を作成することができます。
サンプルの表示 

非表示またはアニメーション付きのペイン

ユーザーが非表示ペインの上にマウスを移動、クリック、アクティブ化、コマンドする場合、表示にアニメーションします。表示されるときに、ユーザーがピン固定できます。
サンプルの表示 

 
 

スプリッターでコンテンツをサイズ変更

ドックされたコンテンツ ペインは、端をドラッグしてサイズ変更することができます。
サンプルの表示 

 レイアウトの保持

ユーザーが設定したレイアウトを XML へ保存し、アプリケーションの起動時に再びレイアウトを読み込むことができます。
サンプルの表示 

 
 

HWND サポート

ホストされる HWND をサポートします。つまり、この WPF ドッキング コントロールーで Windows Forms コントロールをホストできます。
