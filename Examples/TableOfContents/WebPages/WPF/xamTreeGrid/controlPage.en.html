The xamTreeGrid combines the powerful abilities of displaying data in rows and columns with a tree column for expand/collapse hierarchical data. The control is a part of the data presenter family and is essentially a xamDataPresenter that implements a single TreeView without the ability to dynamically switch between views.

Flexible Databinding

The control supports binding to both homogeneous (single type) data source and heterogeneous (multiple types) data source.
View Sample

Configurable and Customizable

The control supports a large amount of configurable options (like tree field location, expansion indicators display mode or selection behavior) and also supports themes and custom styles for different control's UI elements.
View Sample

Filtering

The control supports data filtering with the ability to control the filter action over the filtered out records. You can also configure the visibility of child records which does not match the filtering criteria.
